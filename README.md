# tanghaoming
Интеллектуальная контактная форма с поддержкой ИИ-агента
Он должен добиваться от посетителя набора каких-то данных, например: телефон и емеил и после этого отправлять/сохранять эти данные клиента на указанный емейл. Набор слотов которые надо узнать и емейл должен быть настраиваемым, в отдельном конфиге
